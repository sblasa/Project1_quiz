My Project 1 Quiz

Question 1:

What do we call the style of design that targets small devices by default and scales up to large monitors?

We call this style of design responsive design.

Question 2:

What feature of CSS do we make use of to change the design for different screen sizes?

We call this media queries.

Question 3:

What is the name of the library we use to help with our designs?

The name of the libarary that we use is called Bootstrap.

Question 4:

Describe the grid system in that library.

The grid system is a box that has been divided into 12 columns. With this, you can determine the width based on how wide you want the columns. For example, you can can have a grouping of six columns wide and another six columns which would equal 12 columns, or you can use eight columns wide and four columns wide. You use elements within the containers for these grids.
