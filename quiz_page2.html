Question 6:

Name and describe 4 different kinds of positioning in CSS?

There are four different kinds of positioning in CSS. They are relative, absolute, static, and fixed. Relative: is an offset from its normal position. Static is default flow ignores top and left. Absolute is an offset from the HTML doc. Fixed is an offset from the html doc, and it scrolls underneath it.

Question 13:

What are the four pillars of object oriented programming?

The four pillars of programming are Encapulation, Abstraction, Polymorphism, and Inheritance.

Question 14:

Describe each of the 4 pillars of object oriented programming.

Encapsulation: code is enclosed within a class--usually public and private.

Abstraction: external code that we can't see or know about (private variables and methods). >

Inheritance: allow code reuse and has a parent/child relationship.

Polymorphism: use a child object whereever its parent can be used. This also works with interfaces (but not in JavaScript--currently).

Question 15:
